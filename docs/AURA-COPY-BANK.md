# AURA / ARO — Copy Bank

> The final, high-converting copy for the whole funnel — site, ads, and email.
> Written to a deliberate emotional arc that sequences the **romantic** (craft,
> belonging) before the **operator** (margin, proof), with sales psychology
> baked into each beat.
>
> Name shown as **aro**; swap for **aura** if chosen. Numbers marked
> *(illustrative)* are placeholders — replace with real data before launch.
> The synthesis: **aro** = the name/circle/club; **aura** = the warmth it
> protects.

---

## Voice rules
- Warm, human, confident, quietly witty. Never corporate, never hypey.
- Lead with feeling; back it with numbers. Romance first, proof second.
- Short sentences. One idea per line. Read it aloud — if it sounds like a
  brochure, cut it.
- Talk about *their* café and *their* regulars, not "our platform."
- Never blue/cold language ("solutions", "leverage", "seamless"). Warm verbs:
  remember, welcome, keep, glow, belong, return.

---

## Sales-psychology map (why the page is ordered this way)
1. **Hero** — desire + identity ("your circle").
2. **Social proof** — others like you already trust it.
3. **The leak** — loss aversion (the pain they feel but can't name).
4. **The turn** — hope + mechanism (it's fixable).
5. **Moving where they are** — the magic + a return-rate proof.
6. **How it works** — friction reduction (it's easy, we do it).
7. **The numbers** — operator proof (justify the romance).
8. **Scarcity** — exclusivity (a few per neighbourhood).
9. **Testimonial** — authority + relatability.
10. **Offer** — risk reversal + a single clear action.
