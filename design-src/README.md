# AURA landing — design source

`components/aura/AuraLanding.tsx` is generated by concatenating the modular
design files in `landing/` (atoms, phones, sections-a/b/cozy/c, app) into one
`"use client"` component.

To edit the landing, change the files in `landing/`, then regenerate:

```bash
node design-src/assemble-landing.cjs
```

Styles live in `app/aura-landing.css`. Testimonial photos go in
`public/assets/testimonials/`.
